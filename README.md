# 🔔 ドラえもん人狼 (DORA-WOLF)

### *「入れ替わりロープでシャッフルされた、すすきヶ原の日常。この中に、ガキ大将が潜んでいる。」*
> ドラえもんの世界観をベースにした、対面プレイ専用の人狼ゲーム・Webアプリケーションです。

## 📖 世界観 (Worldview)

ひみつ道具「**入れ替わりロープ**」で遊んでいた、すすきヶ原の子供たち。
しかし、ロープを放した瞬間に全員の姿と中身がめちゃくちゃに入れ替わってしまった！

見た目はいつもの友達だけど、その中身は誰なのか分からない……。
さらに、この混乱に乗じて「ガキ大将」たちが悪巧みを企んでいるようです。

「ドラえもん」や「しずかちゃん」たちの能力を駆使して、子供たちは紛れ込んだガキ大将を見つけ出せるでしょうか？それとも、ガキ大将たちの天下になってしまうのでしょうか？

---

## 🎯 設計思想 (Design Philosophy)

### 1. 「1台のスマホで完結する」ユーザー体験
専用のアプリインストールや複数端末の同期は不要です。1台のスマートフォンを順番に手渡し（パスアラウンド）して進行することを前提としたUI設計を行っています。各プレイヤーの「確認画面」を挟むことで、他のプレイヤーに役職を知られることなくスムーズにゲームが進みます。

### 2. グラフィカルで直感的なUI
ガラス細工のような透明感のある「グラスモルフィズム」デザインを採用。暗い場所でのプレイでも目が疲れにくく、深夜のパーティーシーンを盛り上げるサイバーでプレミアムな視覚演出を提供します。

### 3. 多彩な役職と特殊な勝敗条件
従来の人狼ゲームの枠を超えた、ドラえもんのキャラクターにちなんだユニークな能力を多数搭載。
「自分が誰か分かっていないのび太」や「勝手に怒り出す神成さん」など、カオスな展開が楽しめます。

## 🎭 登場役職 (Roles)

### 👊 人狼陣営 (Bully Side)
- **ジャイアン**: 人狼。村人に紛れて悪さをする。
- **メカジャイアン**: ジャイアンを倒すために作られたロボットだったが、意気投合。村人に紛れて悪さをする。（※村人側の能力も使用可能）
- **スネ夫**: 村人。人狼側についており、裏で暗躍する。しかし人狼はそれを知ることはない。不憫。

### 🔔 村人陣営 (Children Side)
- **ドラえもん**: 村人。町の平和を守るため、することレンズでその人を見通す。人狼なら黒が、それ以外なら白が出る。スネ夫も白が表示される。
- **しずか**: 村人。町の平和を守るため、人狼に襲撃されたけが人を治す。連続で守ることや自身を守ることは不可能。
- **ドラミ**: 村人。町の平和を守るため、テレバしいで今までに気絶してしまった人の役職を知る。余談だが、兄より正確な役職が知れるのは、ひとえにあなたが兄より優秀だから。
- **ジャイ子**: 村人。能力ターンにする行動はない。人狼が殴ってきたとき、あなたは気絶してしまうが、ママが発見することで人狼は返り討ちにあう。
- **セワシ**: 村人。町の平和を守るため、前の日にどんな行動をしていたのかタイムマシンで確認する。中古品のため画質が悪く、その人の役職までは知ることができない。
- **先生**: 村人。家庭訪問のため、人狼村人問わず対象者をその日の行動を足止めする。連続で足止めすることや自身を選択することは不可能。
- **のび太**: 村人。自分が何かしらの役職を持っていると思いこんでいるが、能力の使用結果はでたらめ。自分を「しずか」「先生」だと思いこんでいる場合、ゴルゴンの首の襲撃から守ったり、行動を妨害したりすることができる。なぜならゴルゴンの首を倒したのはあなただから。

### 🎓 第三者陣営 (Third Parties)
- **出木杉**: 第三者。対象者の役職を予想して推理することができる。推理が当たっていれば独自の方法で対象者を気絶させるが、間違っていればショックで自身が気絶する。
- **ゴルゴンの首**: 第三者。自由に動き、人一人を石化する。しずかの守護は貫通するが、しずかや先生だと思いこんでいるのび太の守護や妨害には阻まれる。
- **コピーロボット**: 第三者。一度だけ対象者を気絶させ、役職をコピーする。コピーした後はその役職の勝利条件に従う。のび太をコピーしてものび太が勘違いした役職までコピーするため、あなたはそれに気づかない。
- **はる夫**: 第三者。誰か一人を「安雄」に登録する。この登録は変更できない。自身の生死にかかわらず、安雄がゲーム終了時まで生きていれば勝利となる。
- **神成さん**: 第三者。能力ターンにする行動はない。ランダムで生存している自分以外のプレイヤーを1人ずつ叱りつけて気絶させる。とばっちりもいいところであるため、会話ターンではひそかに「この中に神成さんがいる」との噂が流れている。

## 🔧 ゲームシステム (Systems Design)

### ゲームフロー
1. **Config（設定）**: プレイヤー名を入力し、登場する役職の数を自由にカスタマイズ。
2. **Setup（役職配分）**: ランダムに役職が割り振られ、一人ずつ隠れて確認。
3. **Discussion（議論）**: 制限時間内に誰がガキ大将か話し合う。「神成さん」の噂が流れることも……。
4. **Voting（投票）**: 最も疑わしい人を追放。同票の場合は決選投票へ。
5. **Abilities（能力）**: 夜のターン。一人ずつ端末を受け取り、それぞれの特技を使用。
6. **Result（結果）**: 翌朝、誰が気絶したかが発表される。

## 🕹 操作方法 (Controls)

- **タップ / クリック**: 選択、ボタン操作、アコーディオンの開閉。
- **入力フォーム**: プレイヤー名の追加。
- **スマホ手渡し**: 画面に「〇〇さんに渡してください」と出たら、その人に端末を渡して「確認」を押してもらう。

---

## 🛠 技術スタック (Technical Stack)

- **フロントエンドのみで完結**: サーバーサイド処理不要。
- **Vanilla JS**: 外部ライブラリに頼らない高速なゲームエンジン。
- **localStorage**: 過去の勝率などの統計データをブラウザに保存。
- **CSS Animations**: 滑らかな画面遷移と、カウントダウンタイマーの緊迫感を演出。

## 🚀 実行方法 (Getting Started)

特別なインストールは不要です。

1. このプロジェクトのディレクトリにある `index.html` をブラウザで開く。
2. もしくは、簡易的なHTTPサーバーでホストする：
```bash
# Pythonの場合
python -m http.server 8000
```
3. ブラウザで `localhost:8000` にアクセス。

## 📝 制作メモ (Retrospective)

「人狼ゲームを如何にドラえもんの『日常』に落とし込むか」をテーマに開発。
ジャイ子のカウンターや、のび太の「守っているつもりはないのにゴルゴンの首を倒した設定での耐性」など、原作リスペクトのギミックを盛り込みました。

単なる「人を探すゲーム」ではなく、「入れ替わりロープのせいで自分でも自分が誰だか分かっていない」という混乱が、対面でのコミュニケーションをより加速させることを意図しています。
