# ドラえもん人狼 (Doraemon Werewolf)

1台のスマートフォンを参加者同士で回して遊ぶ、特殊ルールの対面型人狼ゲームです。
ドラえもんの世界観をベースに、原作キャラクターの個性を活かした独自の役職や複雑な能力干渉が特徴のブラウザゲームです。

## 🎮 概要

本ゲームは、Webブラウザ（スマホ推奨）で動作するシングルページアプリケーション（SPA）です。アプリのインストールは不要で、ゲームマスター（GM）なしで手軽にプレイできます。
[cite_start]「○○さんですか？ 今から役職を示します」「○○に代わってください」といった前置きと指示がUIに表示され、1台の端末を回すだけでゲームが進行します [cite: 8]。

## ✨ 特徴

- **GM不要のオート進行**: タイマー管理、能力の処理順序、勝敗判定などはすべてシステムが自動で行います。
- [cite_start]**「のび太」システム**: プレイヤー自身も自分が何者か分からない（他の役職だと思い込んでいるが、能力結果はでたらめになる）という、独自の狂人ポジションを実装しています [cite: 66, 67]。
- **複雑な能力チェーン**: 「先生」による行動阻害や、「コピーロボット」による役職の簒奪など、一般的な人狼にはない特殊な処理順序（プライオリティ）が楽しめます。

## 🔄 ゲームの流れ

[cite_start]ゲームは以下のフェーズ順に進行します [cite: 1]。

1. [cite_start]**Config（設定）**: 役職人数、プレイヤー名、昼の会話時間（デフォルト5分）、初日投票の有無（デフォルトなし）を設定します [cite: 2, 3, 4, 5]。
2. [cite_start]**Game:能力確認**: 登録順にスマホを回し、各自の役職を確認します（初日のみ） [cite: 7, 8]。
3. [cite_start]**Game:会話ターン**: 生存者・気絶者リストと、ゲーム中に存在する役職（ランダム設定含む）がUIに表示されます [cite: 10][cite_start]。時間になるとアラームが鳴ります [cite: 11]。
4. [cite_start]**Game:投票**: 多数決で追放者を決定します。追放された人は、のび太であれば勘違いしていた役職も同時に発表されます [cite: 11]。
5. [cite_start]**Game:能力ターン**: 名前登録順にスマホを回し、各自の能力を使用します [cite: 12, 13][cite_start]。全員の入力完了後に内部処理が行われ、結果が表示されます [cite: 14, 16]。
6. [cite_start]**Judge:判定フェーズ**: 人狼・村人・第三者それぞれの勝利条件を満たしたか判定され、全プレイヤーの正体が明かされます [cite: 17, 18, 19, 20, 21]。

## 🎭 役職一覧

計15種類の役職が存在し、それぞれの陣営の勝利を目指します。

### 🐺 人狼陣営
- [cite_start]**ジャイアン**: 毎晩一人を襲撃します。相方がいる場合はお互いを認識できます [cite: 25, 27]。
- [cite_start]**メカジャイアン**: 村人の能力をコピーして行使するか、通常の襲撃を行うかを選択できます [cite: 29]。
- [cite_start]**スネ夫**: 裏で暗躍しますが、能力ターンに行うことはありません [cite: 31, 32]。

### 🏡 村人陣営
- [cite_start]**ドラえもん（占い師）**: 対象の陣営（黒・白）を見通します [cite: 35, 36]。
- [cite_start]**しずか（騎士）**: 人狼の襲撃から対象を守ります。連続守護や自分自身は守れません [cite: 38, 39]。
- [cite_start]**ドラミ（霊媒師）**: 気絶者の正確な役職を知ることができます [cite: 41, 42]。
- [cite_start]**ジャイ子**: 襲撃してきた人狼を返り討ちにして気絶させます [cite: 44, 45]。
- [cite_start]**セワシ**: タイムマシンで、対象者が前日に「誰に能力を使ったか」を特定します [cite: 47, 48]。
- [cite_start]**先生**: 家庭訪問により、対象者のその日の行動を足止め（無効化）します [cite: 63, 64]。
- [cite_start]**のび太**: 自身を別の役職だと思い込んでおり、能力結果もでたらめになります [cite: 67, 68]。

### 👽 第三者陣営
- [cite_start]**出木杉**: 対象の役職を推理し、当たれば相手を気絶、外せば自分が気絶します [cite: 78, 79]。
- [cite_start]**ゴルゴンの首**: 対象を石化させます。しずかの守護を貫通しますが、唯一「しずかだと思い込んでいるのび太」だけには防がれます [cite: 81, 82]。
- [cite_start]**コピーロボット**: 一度だけ対象者を気絶させ、その役職と勝利条件をコピーします [cite: 84, 85]。
- [cite_start]**はる夫**: 誰か一人を「安雄」に登録し、その人物が最後まで生き残れば勝利します [cite: 87, 88]。
- [cite_start]**神成さん**: 能力ターンにランダムなプレイヤーを1名気絶させます [cite: 90, 91]。
