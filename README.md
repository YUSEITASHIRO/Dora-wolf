# ドラえもん人狼 (Doraemon Werewolf)

ドラえもんの世界観をベースにした、特殊ルール人狼ゲーム。
1台のスマートフォンを参加者全員で回しながらプレイすることを前提に設計されている。GM（ゲームマスター）は不要で、ブラウザのみで動作する。

## 特徴
- **1台のスマホで完結**: アプリをインストールする必要はなく、WebブラウザでURLを開くだけでプレイ可能。
- **特殊な役職**: 「自分の役職を勘違いしているのび太」や、「能力を奪うコピーロボット」など、通常の人狼にはないギミックを多数搭載。
- **自動進行**: 夜の行動の優先度（先生の妨害、しずかの守護、襲撃など）はすべてシステムが自動で処理する。

## ゲームの全体の流れ

ゲームは以下のフェーズ順に進行する。

1. **Config（設定）**: 参加人数、役職の内訳、昼の会話時間などを設定。
2. **Role Confirm（能力確認）**: スマホを回し、各自が自分の役職を確認する。
3. **Talk（会話ターン）**: タイマーを使って議論を行う。「神成さん」がいる場合はアラートが出る。
4. **Night Action（能力ターン）**: 生存者がスマホを回し、個別に夜の能力を使用する。
5. **Night Result（結果表示）**: 夜の行動がシステム内で一斉処理され、朝の犠牲者が発表される。
6. **Judge（判定）**: 勝敗判定。全プレイヤーの正体と戦歴が公開される。

## システム仕様の特記事項

* **のび太の仕様**: 画面上では思い込んでいる役職（例：ドラえもん）として振る舞うが、占いの結果などはランダムな偽情報が表示される。本人はゲーム終了時まで自分がのび太であることに気づけない。
* **セワシの履歴参照**: 先生の妨害を受けていた場合は「×」と表示される。
* **ゴルゴンの首の特殊処理**: ゴルゴンの首に気絶させられたプレイヤーは、ドラミが占っても「不明」と表示される。
